## Описание

`Weather Diary` — это десктопное GUI-приложение для ведения дневника погоды. Оно позволяет пользователям создавать записи о погодных условиях, включая дату, температуру, описание и наличие осадков. Данные можно просматривать в удобной таблице и фильтровать по различным критериям.

### Функциональные возможности:

*   **Добавление записей:** Простая форма для ввода данных о погоде.
*   **Просмотр данных:** Все записи отображаются в таблице, отсортированной по дате от самой новой к самой старой.
*   **Фильтрация:**
    *   По конкретной дате.
    *   По температуре (отображение записей с температурой выше указанного значения).
*   **Сохранение данных:** Все записи автоматически сохраняются в файл `weather_data.json` и загружаются при запуске приложения.
*   **Валидация ввода:** Приложение проверяет корректность вводимых данных (формат даты, числовое значение температуры).

## Технологии

*   **Язык:** Python 3
*   **GUI:** `tkinter` (стандартная библиотека) и `tkcalendar` (для выбора даты).
*   **Формат данных:** JSON
*   **Система контроля версий:** Git

## Как запустить

1.  **Клонирование репозитория:**
    Склонируйте репозиторий или скачайте все файлы проекта в одну папку.

2.  **Установка зависимостей:**
    Для работы календаря в приложении требуется библиотека `tkcalendar`. Установите её с помощью `pip`:
    ```bash
    pip install tkcalendar
    ```

3.  **Запуск приложения:**
    Откройте терминал в директории с проектом и выполните команду:
    ```bash
    python weather_diary.py
    ```

## Примеры использования

*   **Добавление записи:**
    1.  Выберите дату в календаре.
    2.  Введите температуру (например, `15.5`).
    3.  Введите описание (например, `Солнечно, небольшой ветер`).
    4.  Выберите, были ли осадки ("Да"/"Нет").
    5.  Нажмите кнопку "Добавить". Новая запись появится вверху таблицы.

*   **Фильтрация по дате:**
    1.  В секции "Фильтрация" выберите интересующую вас дату.
    2.  Нажмите "Применить фильтр". В таблице останутся только записи за этот день.

*   **Фильтрация по температуре:**
    1.  В поле "Температура выше (°C)" введите пороговое значение (например, `10`).
    2.  Нажмите "Применить фильтр". Таблица покажет дни, когда температура была выше 10°C.

*   **Сброс фильтров:**
    Нажмите кнопку "Сбросить все", чтобы снова отобразить все записи в дневнике.
